## Overview

Ghibli Films is a small iOS app built with SwiftUI for browsing Studio Ghibli movies.
The app uses [Studio Ghibli API](https://ghibliapi.vercel.app/) for fetching films, it might be unable to use in some countries.

## Features

- Browse a list of Studio Ghibli films
- Open a detailed screen for each film
- View film metadata such as title, director, producer, release year, running time, and rating
- Search films by title
- Add and remove films from favorites
- Persist favorites locally
- View all favorite films on a separate screen
- Change the app appearance with a light mode setting
- Use mock and sample data for previews and testing

## How to Run

1. Open `GhibliSwiftUiApp.xcodeproj` in Xcode.
2. Select the `GhibliSwiftUiApp` scheme.
3. Choose an iPhone simulator or a connected iPhone.
4. Press `Cmd + R` to build and run the app.
